# VianTech - Landing Page Perusahaan Teknologi

Ini adalah proyek landing page statis untuk **VianTech**, sebuah perusahaan fiktif yang berfokus pada penyediaan perangkat digital dan solusi hiburan interaktif. Landing page ini dirancang untuk menjadi modern, bersih, dan sepenuhnya responsif.

Proyek ini dibuat menggunakan **Tailwind CSS** untuk styling yang cepat dan efisien.

## âœ¨ Fitur Utama

Berdasarkan file `index.html`, halaman ini memiliki beberapa bagian utama:

* **Navigasi:** Navbar yang *sticky* dan responsif, lengkap dengan menu *mobile* (hamburger) menggunakan JavaScript sederhana.
* **Hero Section:** Bagian *headline* utama dengan gradien latar belakang yang menarik.
* **Kategori Produk:** Menampilkan tiga kategori unggulan:
    * Laptop (menggunakan `laptop.jpg`)
    * PC Games Original (menggunakan `game.jpg`)
    * Konsol & Aksesoris (menggunakan `ps5.webp`)
* **Tentang Kami:** Deskripsi singkat mengenai VianTech.
* **Kontak:** Berisi formulir kontak sederhana untuk *leads*.
* **Footer:** Menyertakan informasi hak cipta dan ikon media sosial dari Font Awesome.

## ğŸ’» Teknologi yang Digunakan

Proyek ini dibangun menggunakan teknologi web standar:

* **HTML 5**: Untuk struktur halaman web.
* **Tailwind CSS**: (Dimuat via CDN) Untuk semua styling dan tata letak responsif.
* **Font Awesome**: (Dimuat via CDN) Untuk ikon media sosial di footer.
* **JavaScript**: (Inline) Untuk fungsionalitas *toggle* menu mobile.
